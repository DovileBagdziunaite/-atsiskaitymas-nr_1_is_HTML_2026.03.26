🌿 Prieskonių pasaulis

    Trumpas edukacinis tinklalapis apie prieskonius iš viso pasaulio. Projektas sukurtas naudojant paprastą HTML ir skirtas susipažinti su puslapio struktūra, navigacija bei formomis.

📖 Aprašymas

    Ši svetainė pateikia informaciją apie įvairius prieskonius, jų naudą ir panaudojimą virtuvėje. Projektą sudaro keli puslapiai:

    Pagrindinis puslapis – bendras pristatymas apie prieskonių svarbą
    Prieskonių puslapis – populiarių prieskonių aprašymai
    Kontaktų puslapis – forma susisiekti su svetainės kūrėjais

⚙️ Naudotos technologijos

    HTML5
    Inline CSS (stiliai tiesiogiai elementuose)

✨ Funkcionalumas

    Navigacijos meniu tarp puslapių
    Paieškos laukelis
    Prisijungimo forma
    Kontaktų forma
    Paprastas responsyvus išdėstymas naudojant Flexbox

📌 Tikslas

    Projektas skirtas atsiskaitymui nr.1 – suprasti HTML struktūrą, puslapių kūrimą ir pagrindinius UI elementus.
